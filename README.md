#  Предсказание стоимости жилья (California Housing)

Этот проект — модель машинного обучения, которая предсказывает медианную стоимость дома в округе (block group) в Калифорнии. Проект основан на данных переписи населения 1990 года.

Основная задача: **Регрессия** (предсказание числового значения).
Используемый алгоритм: **Линейная регрессия**.

##  Описание данных

В датасете содержатся следующие признаки (features):
- **Longitude / Latitude**: Географические координаты.
- **Housing Median Age**: Средний возраст домов в округе.
- **Total Rooms / Bedrooms**: Общее количество комнат и спален.
- **Population**: Население округа.
- **Households**: Количество домохозяйств.
- **Median Income**: Медианный доход жителей (очень важный признак).
- **Ocean Proximity**: Близость к океану (категориальный признак).

**Целевая переменная (Target):** `median_house_value` (стоимость дома).

##  Технологии

Проект написан на **Python** с использованием стека для Data Science:

*   **Jupyter Notebook** — интерактивная среда разработки.
*   **Pandas** — загрузка данных и манипуляции с таблицами.
*   **NumPy** — матричные вычисления.
*   **Matplotlib / Seaborn** — визуализация данных и зависимостей.
*   **Scikit-Learn** — библиотека машинного обучения (предобработка данных, LinearRegression, метрики).

##  Предобработка данных (Pipeline)

Перед обучением модели данные проходят очистку:
1.  **Заполнение пропусков:** В столбце `total_bedrooms` пропущенные значения заменяются медианой (используется `SimpleImputer`).
2.  **Работа с признаками:**
    - Создание новых признаков (например, отношение кол-ва комнат к кол-ву домохозяйств).
    - Удаление или трансформация категориальных признаков (`ocean_proximity`).
3.  **Масштабирование:** Приведение числовых данных к единому масштабу (StandardScaler).

##  Как запустить проект

Для запуска проекта вам понадобится Python и установленные библиотеки. Рекомендуется использовать виртуальное окружение.

1.  **Клонируйте репозиторий:**
    ```bash
    git clone <ссылка-на-ваш-репозиторий>
    cd ml
    ```

2.  **Активируйте виртуальное окружение:**
    *   Windows:
        ```bash
        my_env\Scripts\activate
        ```
    *   macOS/Linux:
        ```bash
        source my_env/bin/activate
        ```

3.  **Установите зависимости (если есть requirements.txt):**
    ```bash
    pip install -r requirements.txt
    ```
    *Или установите вручную:*
    ```bash
    pip install pandas numpy matplotlib scikit-learn jupyter
    ```

4.  **Запустите Jupyter Notebook:**
    ```bash
    jupyter notebook
    ```
    Откройте файл `Housing.ipynb`.
